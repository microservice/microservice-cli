# `oms subscribe` example
The [slack](https://github.com/microservice/slack) microservice is used for this example.

## Subscribing to an event
### Input
```
oms subscribe bot hears -a channel="CAL6YMP9C" -e BOT_TOKEN="xoxb-*********************"
```
